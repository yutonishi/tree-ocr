# tree-ocr
Google Cloud Vision AIでツリー構造の情報を読み取るサンプルスクリプト

## 実行手順
- .envファイル作成
```
GOOGLE_CLOUD_API_KEY=xxx
```

- 下記の手順で実行
```
# openai_envという名前で仮想環境を作成（同名のディレクトリが作成される）
python3 -m venv tree-ocr_env

source tree-ocr_env/bin/activate

pip install 必要はものたち

# 実行
python3 request.py
```
